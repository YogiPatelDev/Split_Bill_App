# Split Bill App

## Project Description
The Split Bill App is designed to simplify the process of splitting bills among friends, family, or colleagues. Users can input their expenses and the app calculates each person's share, making group outings, dinners, or trips hassle-free.

## Features
- **User-Friendly Interface:** Easy to navigate and input expenses.
- **Customizable Bill Splitting:** Ability to add multiple people and customize their share of the bill.
- **Expense Tracking:** Users can keep a record of past expenses for future reference.
- **Multi-Currency Support:** Ability to handle different currencies for international users.
- **Share Results:** Users can easily share the final calculations via text or social media.

## How to Use It
1. **Input Expenses:** Enter the total amount and specify who participated in the expense.
2. **Add Users:** Add the names of individuals sharing the bill.
3. **Calculate Shares:** Click on the calculate button to view how much each person owes or will receive.
4. **Review Past Expenses:** Navigate to the history section to see previous calculations and expenses.

## Installation
1. Clone the repository: `git clone https://github.com/YogiPatelDev/Split_Bill_App.git`
2. Navigate to the project directory: `cd Split_Bill_App`
3. Install required packages: `npm install`
4. Run the application: `npm start`
